# Egzersiz Görselleri

Bu klasör egzersiz rehber görsellerini içerir.

## Görsel Kategorileri

- **thumbnail**: Egzersiz küçük resmi (liste görünümü için)
- **step1, step2, step3**: Egzersizi nasıl yapacağınızı gösteren adım adım görseller
- **demo**: Egzersizin tamamlanmış hali
- **form**: Doğru form ve pozisyon görseli

## Kullanım

Görseller veritabanındaki `exercise_images` tablosunda saklanır ve her egzersizle ilişkilendirilir.

## Format

- PNG veya JPG formatında
- Önerilen boyut: 800x600 piksel
- Dosya adı formatı: `exercise_name_type.png` (örn: `bench_press_step1.png`)

## Placeholder

Gerçek görseller eklenmeden önce placeholder görseller kullanılır.
